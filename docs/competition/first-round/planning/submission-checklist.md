# Round 1 Submission Checklist

## A) Core Deliverables
- [ ] MCQ answers completed (10 questions)
- [ ] Kaggle submission uploaded (Part 3)
- [ ] Report PDF finalized (NeurIPS format, <= 4 pages excluding refs/appendix)
- [ ] GitHub repository link included and accessible

## B) Report Content Requirements
- [ ] Part 2 visualizations are clear, labeled, and readable
- [ ] Analysis includes Descriptive, Diagnostic, Predictive, Prescriptive layers
- [ ] Part 3 modeling pipeline is documented
- [ ] Explainability section included (feature importance/SHAP/PDP equivalent)
- [ ] Leakage prevention and validation strategy described

## C) Competition Constraints
- [ ] No external data used
- [ ] Results are reproducible from repository code
- [ ] Submission format matches `sample_submission.csv`

## D) Admin Requirements
- [ ] Student ID photos from all team members prepared
- [ ] Final-round attendance confirmation requirement satisfied
- [ ] Official form fields are fully completed
- [ ] All links in form are correct and accessible

## E) Final Quality Gate
- [ ] Last full read-through performed for report and README
- [ ] Repository cleaned of dead files and broken references
- [ ] Deadline buffer reserved for upload issues
